# 08. 배열 내장함수
### forEach
* 배열 안에 있는 원소들에 대해 일관된 작업을 하고 싶을 때 사용한다.
* 예외를 발생시키지 않으면 중간에 반복을 종료할 수 없다.
```{.javascript}
const superheroes = ['아이언맨', '캡틴 아메리카', '토르'];

function print(hero) {
    console.log(hero);
}

superheroes.forEach(print);
```

### map
* 배열 안에 있는 원소들을 전체적으로 변환하고 싶을 때 사용한다.
```{.javascript}
const array = [1, 2, 3, 4, 5];

const square = n => n * n;
const squared = array.map(square);

console.log(squared);       //[ 1, 4, 9, 16, 25 ]
```
```{.javascript}
const items = [
    {
        id: 1,
        text: 'hello'
    },
    {
        id: 2,
        text: 'bye'
    }
];

const texts = items.map(item => item.text);
console.log(texts);         //[ 'hello', 'bye' ]
```

### indexOf
* 일반 배열에서 조건을 만족하는 가장 첫번째 원소의 위치를 찾는다.
```{.javascript}
const superheroes = ['아이언맨', '캡틴 아메리카', '토르'];

const index = superheroes.indexOf('토르');

console.log(index);         //2
```

### findIndex
* 배열 안에 내용물이 객체일때, 조건을 만족하는 가장 첫번째 원소의 위치를 찾는다.
```{.javascript}
const todos = [
    {
        id: 1,
        text: '자바스크립트 입문',
        done: true,
    },
    {
        id: 2,
        text: '함수 배우기',
        done: true,
    },
    {
        id: 3,
        text: '객체와 배열 배우기',
        done: true,
    },
    {
        id: 4,
        text: '배열 내장함수 배우기',
        done: false,
    }
];

const index = todos.findIndex(todo => todo.id === 3)
console.log(index);         //2
```

### filter
* 특정 조건을 만족하는 원소들을 찾아서 새로운 배열을 만든다.
```{.javascript}
const todos = [
    {
        id: 1,
        text: '자바스크립트 입문',
        done: true,
    },
    {
        id: 2,
        text: '함수 배우기',
        done: true,
    },
    {
        id: 3,
        text: '객체와 배열 배우기',
        done: true,
    },
    {
        id: 4,
        text: '배열 내장함수 배우기',
        done: false,
    }
];

const tasksNotDone = todos.filter(todo => !todo.done);
console.log(tasksNotDone);          //[ { id: 4, text: '배열 내장함수 배우기', done: false } ]
```

### splice
* 기존 배열에서 특정 원소를 제거할 때 사용한다.
```{.javascript}
const numbers = [10, 20, 30, 40];
const index = numbers.indexOf(30);
const spliced = numbers.splice(index, 2); // index 포함 2개 제거.

console.log(spliced);       //[ 30, 40 ]
console.log(numbers);       //[ 10, 20 ]
```

### slice
* 배열에서 특정 원소를 잘라낼 때 사용한다. 
* 기존 배열에 영향을 주지 않는다.
```{.javascript}
const numbers = [10, 20, 30, 40];
const sliced = numbers.slice(0, 2); // 0부터 2앞까지 제거.
console.log(sliced);        //[ 10, 20 ]
console.log(numbers);       //[ 10, 20, 30, 40 ]
```

### shift
* 배열의 첫번째 원소를 기존 배열에서 추출한다.
```{.javascript}
const numbers = [10, 20, 30, 40];
const value = numbers.shift();
console.log(value);         //10
console.log(numbers);       //[ 20, 30, 40 ]
```

### pop
* 배열의 마지막 원소를 기존 배열에서 추출한다.
```{.javascript}
const numbers = [10, 20, 30, 40];
const value = numbers.pop();
console.log(value);         //40
console.log(numbers);       //[ 10, 20, 30 ]
```

### unshift
* 기존 배열의 맨앞에 원소를 추가한다.
```{.javascript}
const numbers = [10, 20, 30, 40];
numbers.unshift(5);
console.log(numbers);       //[ 5, 10, 20, 30, 40 ]
```

### push
* 기존 배열의 맨뒤에 원소를 추가한다.
```{.javascript}
const numbers = [10, 20, 30, 40];
numbers.push(50);
console.log(numbers);       //[ 10, 20, 30, 40, 50 ]
```

### concat
* 배열을 합치는데 사용한다
* 기존의 배열에 영향을 주지 않는다.
```{.javascript}
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

const concated = arr1.concat(arr2);     //(=[...arr1, ...arr2];)
console.log(concated);                  //[ 1, 2, 3, 4, 5, 6 ]
```

### join
* 배열안의 원소들을 합쳐 하나의 문자열을 만든다.
```{.javascript}
const array = [1, 2, 3, 4, 5];
console.log(array.join());              //1,2,3,4,5
console.log(array.join(' '));           //1 2 3 4 5
console.log(array.join(', '));          //1, 2, 3, 4, 5
```
### reduce
* 배열안에 있는 모든 값들을 연산할 때 유용하다.
```{.javascript}
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((accumulator, current) => accumulator + current, 0) // 누적값, 현재, 계산식, 초깃값
console.log(sum);           //15 
```
