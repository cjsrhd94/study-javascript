# 06. 배열
## 배열
* 배열은 인덱스를 가지는 복수의 자료를 저장할 수 있는 자료구조이다.  
* 데이터는 순차적으로 저장되며 중복이 가능합니다.  
* 저장된 데이터에는 인덱스를 이용하여 접근이 가능합니다.

### 배열 선언 방법
```javascript
const array = new Array();                      // 빈 배열 선언
const array = new Array("a", "b", "c", "d");
const array = new Array(3)                      // 배열 크기 지정
const array = [];                               // 빈 배열 선언(권장)
const array = [a, b, c, d];
const array = [,,,];                            // 배열 크기 지정
```
### 자바스크립트 배열의 특징
* 배열 내부의 데이터 타입이 서로 다를 수 있다.
* 배열의 크기는 동적으로 변경 될 수 있다.

```javascript
const array = [1, 'blabla', {name: '멍멍이'}, 4, true]
array[7] = "apple"

console.log(array);
console.log(array[5]);                                  // undefined
```
