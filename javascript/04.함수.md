# 04. 함수
## 1. 함수
* 함수란 하나의 특별한 목적의 작업을 수행하도록 설계된 독립적인 블록을 의미한다.
* 함수는 필요할 때마다 호출하여 해당 작업을 반복해서 수행할 수 있다.
```{.javascript}
function add(a, b){
    return a + b;
}

const sum = add(1, 2);

console.log(sum);
```
* 템플릿 리터럴
```{.javascript}
function hello(name) {
return('Hello,' + name + '!' )
return(`Hello, ${name}!`)
}

console.log(hello('velopert'));
```

## 2. 화살표 함수

```{.javascript}
const add = (a, b) => {
    return a + b;
}

const sum = add(1, 2);

console.log(sum);
```
