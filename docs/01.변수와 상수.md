# 01. 변수와 상수
## 1. 변수
* 변수(variable)는 데이터를 담기 위한 메모리 공간을 의미한다.
* 자바스크립트에서 변수를 선언할 때는 `let`을 사용한다.(ES6)
* `let`문으로 변수를 선언한 후 해당 변수에 다른 값을 재할당할 수 있다.
* `let`의 scope는 'block'단위이다.
```{.javascript}
let value = 1;  
console.log(value); // 1
value = 2;  
console.log(value); // 2
```

* 변수는 똑같은 이름으로 재선언할 수 없다.  
```{.javascript}
let value = 1;  
let value = 2; // error
```
## 2. 상수
* 상수를 선언할 때는 `const`를 사용한다.  
* 상수는 한번 선언하면 값을 바꿀 수 없다.
* `const`의 scope는 'block'단위이다.
```{.javascript}
const a = 1;
a = 2;      // error
```
## 3. 자바스크립트의 자료형
### 1. 숫자열
```{.javascript}
let value = 1;
```
### 2. 문자열
* 문자열에서 작은 따옴표와 큰 따옴표는 같다.
```{.javascript}
let text = 'hello';
let name = "안녕!";
```


### 3. 논리형(boolean)
```{.javascript}
let good = true;
let bad = false;
```

### 4. `null`은 값이 비어있음을 뜻한다.
```{.javascript}
let friend = null;
```

### 5. `undefined`은 값이 아직 할당되지 않았다는 것을 뜻한다.
```{.javascript}
let criminal; // undefined
```
