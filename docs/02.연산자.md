# 02. 연산자  
## 연산자
* 연산자는 특정 연산을 할 때 사용하는 문자이다.

### 1. 산술연산자  
```{.javascript}
let a = 1 + 2;  // 3  
let b = 5 - 3;  // 2
let c = 3 * 4;  // 12
let d = 6 / 4;  // 1.5 (나누기)
let e = 8 % 3;  // 2 (나머지)
```
* `+`연산자로 문자열도 붙일 수 있다.
```{.javascript}
const a = '안녕';
const b = '하세요';
console.log(a + b);   // 안녕하세요
```

### 2. 증감연산자
* `a++`은 자신을 먼저 1증가 시키고 다른 연산을 시작한다.
* `++a`는 다른 연산을 한 후 자신을 1 증가시킨다.
```{.javascript}
let a = 1;

console.log(a++);   // 1
console.log(a);     // 2
console.log(++a);   // 3

console.log(a--);   // 3
console.log(a);     // 2
console.log(--a);   // 1
```

### 3. 대입연산자
* 대입 연산자는 기본적으로 오른쪽에 있는 값을 왼쪽에 있는 변수에 저장한다.
* 복합 대입연산자는 연산 실행 후 대입을 하는 연산자이며, 피연산자들의 결합 방향은 오른쪽에서 왼쪽이다.
```{.javascript}
let a = 1;

a += 5;    // 6
a -= 4;    // 2
a *= 3;    // 6
a /= 2;    // 3
```

### 4. 논리연산자
* 논리 연산자는 참인지 거짓인지를 나타내어 준다.
* not `!` true와 false를 반전시킨다.  
* and `&&` 둘 중 하나가 false라면 결과값은 false이다.  
* or `||` 둘 중 하나가 true라면 결과값은 true이다.
```{.javascript}
 let a = true;
 let b = false;
 
 console.log(!a)        // false
 console.log(a && b);   // true
 console.log(a || b);   // true
```
* not ➞ and ➞ or 순서로 우선순위가 적용된다.  
```{.javascript}
 let a = true;  
 let b = false;
 
 console.log(!(a && b || a && b));   // true
 ```
### 5. 비교연산자
* 두 값을 비교할 때 사용한다.
```{.javascript}
const a = 1;
const b = 1;

const equals = a === b;
console.log(equals);        // true
const notEquals = a !== b;
console.log(notEquals);     // false
```

* `==`와 `===`의 차이  
`===`는 타입까지 고려하는 반면, `==`는 타입을 고려하지 않는다. 
```{.javascript}
let a = true;
let b = false;

console.log(a == 1);    // true
console.log(b == 0);    // true

let c = null;
let d;

console.log(c == d);    //true

console.log(1 == '1');  //true
```
### 6. 부등호연산자
```{.javascript}
const a = 10;
const b = 15;
const c = 15;

console.log(a < b);     // true
console.log(a <= c);    // true
console.log(b < c);     // false
```
