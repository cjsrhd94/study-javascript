# 03. 조건문
## 조건문
* 특정 조건 만족 시 어떤 작업을 수행하고 싶을 때 사용한다.  
### 1. if, else if, else문
* `if` 안의 조건문이 참이면, `if`문을 실행한다.
```javascript
const a = 1;

if (a + 1 === 2) {
    console.log('a + 1이 2입니다.');
}
```
* `if` 안의 조건문이 참이 아니라면, `else`문을 실행한다.
```javascript
const a = 10;

if (a > 15){
    console.log('a가 15보다 큽니다.');
} else {
    console.log('a가 15보다 크지 않습니다');
}
```
* `elif`문으로 조건을 추가할 수 있으며, 갯수 제한은 없다.
```javascript
const a = 11;

if (a === 5) {
    console.log('5입니다!');
} else if (a === 10) {
    console.log('10입니다!');
} else if (a === 7) {
    console.log('7입니다!');
} else {
    console.log('5도 아니고 10도 아닙니다. 7도 아니네요.');
}
```

### 2. switch case문
* 특정 값에 따라 다른 작업을 할 때 사용한다.
* `default`문은 모든 조건에 해당되지 않을 때 실행된다.
```javascript
const device = 'galaxy tab';

switch (device) {
    case 'iphone':
        console.log('아이폰');
        break;
    case 'ipad':
        console.log('아이패드');
        break;
    case 'galaxy note':
        console.log('갤럭시 노트');
        break;
    default:
        console.log('모르겠네요');
}
```
