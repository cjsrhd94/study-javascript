# 07. 반복문
## 반복문
### 1. for문
* `for`문은 조건식이 `true`일 경우 계속해서 반복한다.
* 주로 반복횟수가 정해진 경우 사용한다.
```{.javascript}
for (let i = 0; i < 10; i++) {
    console.log(i);
};
```
* 아래 예시와 같이 배열과 함께 많이 사용한다.
```{.javascript}
const names = ['멍멍이', '야옹이', '멈뭄이'];

for (let i = 0; i < names.length; i++) {
    console.log(names[i]);
};
```

### 2. while문
* `while`문은 조건식이 `true`일 경우 계속해서 반복한다.
* 주로 무한루프나 특정 조건에 만족할 때까지 반복해야 하는 경우 사용한다.
* 파일을 읽고 쓰기에 많이 사용한다.
```{.javascript}
let i = 0;

while (i < 10) {
    console.log(i);
    i++;
};
```
### 3. do-while문
* 무조건 중괄호 블럭을 한번 실행하고, 조건을 검사하여 반복을 결정한다.
```{.javascript}
let i = 0;

do {
    console.log(i);
    i++;
} while (i < 10);
```


### 4. continue문와 break문
* 반복문에서 `continue`문을 만날시, 조건식으로 이동하여 다음 작업을 계속한다.
* 반복문에서 `break`문을 만날시, 해당 반복문을 종료한다.
```{.javascript}
for (let i = 0; i < 10; i++) {
    if (i === 2) {
        continue;
    }
    console.log(i)

    if (i === 5) {
        break;
    }
};
```
